Web stranica Fitness++

Tijekom stručne prakse sam učio web development. Kroz 10 radnih dana stručne prakse sam izradio ovu web stranicu za izmišljenu fiitnes aplikaciju koja se zove Fitness++. Stranica je izrađena pomoću Bootstrap-a, te je zbog toga responzivna. Stranica sadrži neke Bootstrap elemente poput navigacije, carousel-a, cards i gumba. Koristene su ikone sa stranice FontAwesome. HTML i CSS kod su pisani u programu Visual studio code.
